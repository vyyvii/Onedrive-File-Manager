## 2. Description du service

### 2.1 Fonctionnalités
OneDrive Duplicate Finder est une application logicielle qui :
- Analyse les fichiers photos et vidéos stockés sur Microsoft OneDrive
- Détecte les doublons par critères exacts (nom, taille, hash) ou similarité visuelle
- Permet la comparaison visuelle et la suppression sélective des doublons
- Fournit des statistiques sur les fichiers analysés

### 2.2 Accès aux données
L'Application accède à votre compte OneDrive via l'API Microsoft Graph avec votre autorisation explicite. Les permissions requises sont :
- `Files.Read.All` : Lecture de tous vos fichiers OneDrive
- `Files.ReadWrite.All` : Modification et suppression de vos fichiers OneDrive
- `User.Read` : Lecture de vos informations de profil de base
